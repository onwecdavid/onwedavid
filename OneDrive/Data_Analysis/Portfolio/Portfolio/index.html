Web Scrapping with Python

This code performs web scraping on the job posting website shine.com to extract job details such as job title, URL, company, location, job type, posting date, and required work experience. It does this by constructing URLs with a given job title and location, navigating through multiple pages of search results, parsing the HTML content using BeautifulSoup, and saving the extracted data to a CSV file. The main goal is to increase the efficiency of sourcing job vacancies for a recruitment agency.
